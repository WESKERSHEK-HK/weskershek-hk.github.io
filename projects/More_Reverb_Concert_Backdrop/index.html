• More Reverb - Live Concert Backdrop

    Audio Visual 2019

  • Client
    More Reverb(HK)

  • Role
    Generative Artist

I created six animated backdrops for the Hong Kong band More Reverb.

These visuals acted as live-synced projections during the band's concerts. Each animation reacted in real-time to different instruments in the music.

By utilizing creative coding techniques, I programmed the animations to sync dynamically with the audio waveform in real-time. This allowed the visuals to move and shift coincident with the band's musical performances. Audiences could experience a fully integrated audiovisual show.
