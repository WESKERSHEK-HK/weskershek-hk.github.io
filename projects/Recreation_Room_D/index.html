As an Embedded Software Engineer, I had the privilege of contributing to the Re-create Room D project, an innovative technical art exhibition that recreated the vibrant atmosphere of youth centers from the 90s. This captivating exhibition featured nostalgic game machines like table football and Novus, meticulously reconstructed using mechanical systems.

In my role, I was responsible for designing the electronic schematics and control systems for both the football machine and Novus machine. Leveraging my expertise in embedded Linux, embedded C, ROS Melodic, ESP32, and STM32, I developed a seamless networking communication system that connected these physical machines to a virtual PC game built with Unreal Engine 4.

A significant challenge I encountered was integrating the physical machines with the virtual game, requiring the implementation of robust control and motion tracking systems. Through my embedded software development skills, I successfully created an immersive experience where visitors could interact with the mechanical game machines through the virtual PC game.

The impact of the Re-create Room D exhibition was profound, as it provided visitors with a nostalgic journey back to their youth. The combination of the physical game machines and the virtual game created a truly captivating and engaging experience. I am proud to have played a role in a project that brought joy and nostalgia to countless individuals.
